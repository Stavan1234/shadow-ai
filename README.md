# S.H.A.D.O.W.
### **S**emantic **H**ierarchical **A**utonomous **D**irected **O**ntological **W**eaver

S.H.A.D.O.W. is a lightweight, privacy-centric, 100% offline agentic desktop assistant inspired by the silent, invaluable helpers in the light novel *Shadow Slave*. 

Designed to overcome severe hardware constraints on standard consumer laptops (16GB RAM, no dedicated GPU), it bypasses traditional RAG text bloating by executing a unique **Two-Mind Memory Architecture**:

*   **Conscious Mind:** A blazing-fast, local file-backed memory layout that tracks active session turns instantly.
*   **Subconscious Mind:** A deep, persistent long-term knowledge network powered by **Cognee** (`ladybug` graph traversal and `lancedb` vector embeddings) that runs asynchronous background distillation without lagging conversational chat loops.
